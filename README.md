# HeGCL
Source code for "HeGCL: Advance Self-Supervised Learning in Heterogeneous Graph-level Representation"

## Important Dependencies

```

dgl-cu101==0.6.1
h5py==3.5.0
networkx==2.3
ogb==1.3.1
pandas==1.2.5
scikit-learn==0.23.2
```
